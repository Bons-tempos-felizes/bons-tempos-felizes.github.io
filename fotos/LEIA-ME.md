# Pasta de Fotos da Turma — Bons Tempos Felizes

Aqui estão organizadas as fotos dos encontros anuais, separadas por ano.

## Estrutura

```
fotos/
├── 2017/   ← Encontro de 2017
├── 2018/   ← Encontro de 2018
├── 2019/   ← Encontro de 2019
├── 2020/   ← Encontro de 2020
├── 2021/   ← Encontro de 2021
├── 2022/   ← Encontro de 2022
├── 2023/   ← Encontro de 2023
├── 2024/   ← Encontro de 2024
├── 2025/   ← Encontro de 2025
└── 2026/   ← Encontro de 2026 (próximo, dia 01/05)
```

## Como adicionar fotos

1. Salve as fotos no computador (do celular, da câmera, do WhatsApp etc.)
2. Coloque cada foto na pasta do ano correspondente
3. Pode renomear as fotos como quiser, por exemplo:
   - `01-grupo-todos.jpg`
   - `02-anete-tadeu.jpg`
   - `03-mesa-do-jantar.jpg`

## Para fazer aparecer no site

Depois de adicionar as fotos numa pasta de ano, é só:
1. Subir a pasta do ano para o GitHub (igual fizemos com o `index.html`)
2. Me chamar que eu atualizo a galeria do site para mostrar as fotos
